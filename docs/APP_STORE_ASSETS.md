#### Step 2: First Launch (Apple Watch)

1. Launch PetTracker on Apple Watch

   - Tap "Allow" for Workout data

3. Main screen displays:
   - "Start Tracking" button (large, prominent)
   - GPS indicator: Gray (not tracking)
   - Battery level: Current Watch battery

- UI is responsive and clear

#### Step 3: Start Tracking (Watch)

1. Tap "Start Tracking" button on Watch
2. HealthKit workout session starts (outdoor walk activity)
3. GPS acquisition begins:
   - GPS indicator: Yellow (acquiring signal)
   - Status: "Acquiring GPS..."
   - **Note**: Can take 30-60 seconds outdoors
4. Once GPS lock acquired:
   - GPS indicator: Green (tracking active)
   - Status: "Tracking active"

   - Battery percentage updates

- GPS acquisition faster outdoors with clear sky view

- No crashes or errors

---

#### Step 4: Verify Data Transmission (iPhone)

1. Return to iPhone (Watch continues tracking)
2. iPhone displays:
   - Pet location: Coordinates from Watch GPS
   - Distance: Calculated distance between iPhone and Watch
   - Battery: Watch battery percentage
   - Connection status: "Connected" (green)
   - Location history: GPS fixes accumulate (up to 100)

**Expected Behavior**:
- Location updates appear within 1-2 seconds
- Distance updates in real-time
- Battery level matches Watch display
- No errors or connection issues

**Troubleshooting**:

- No data loss
