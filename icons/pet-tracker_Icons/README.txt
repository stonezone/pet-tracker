pawWatch Icon Set
=====================
Contents:
- iOS/AppIcon.appiconset: App icon set for iOS (iPhone + iPad) including marketing 1024x1024. PNG, sRGB, opaque.
- watchOS/AppIcon.appiconset: App icon set for watchOS including marketing, Home Screen, Notification Center, Short Look, and companion iPhone settings sizes.

Install:
1) Drag the appropriate AppIcon.appiconset folder(s) into your project's *.xcassets (or drop the whole iOS/ or watchOS/ folder).
2) In Xcode target settings, under App Icons and Launch Images, set AppIcon to this set for each target (iOS app target and Watch app target).

Notes:
- Input art was generated at 1024x1024 and downsampled with high-quality resampling.
- All files have no alpha channel (fully opaque) per App Store guidance.
